# Index Domande

Raccoglie tutte le liste di domande usate per testare la preparazione all'esame.

- [[domande_note_validate]]: Tutte le 85 domande chiuse e aperte tratte dagli appelli passati (ex `Domande IUM.pdf`), con risposte verificate e basate esclusivamente sulla Knowledge Base.
- [[domande_appunti_ium_2021]]: Domande generate in via esplorativa dagli appunti 2021.
- [[domande_appunti_collettivi_2025_2026]]: Domande generate specificatamente per i concetti introdotti negli appunti più recenti (Effetto Cobra, Item invertiti, ecc.).
- Le vecchie liste (es. domande_aperte, domande_chiuse) sono parzialmente superate dalla validazione unificata in `domande_note_validate`.
