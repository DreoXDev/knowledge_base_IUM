# Domande Note

Questa cartella raccoglie domande d'esame note o ricorrenti.

Le risposte non sono considerate affidabili finche non vengono validate.

## Stati ammessi

- `VALIDATA`
- `PROBABILE`
- `DA VERIFICARE`
- `POTENZIALMENTE ERRATA`

## Formato

Usare `templates/question_note_template.md` come riferimento.
