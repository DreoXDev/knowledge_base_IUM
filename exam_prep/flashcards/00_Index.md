# Index Flashcards

Raccoglie tutte le risorse a scheda (flashcard) create per il ripasso attivo (Spaced Repetition).

- [[IUM_domande_flashcards]]: Set completo di 85 flashcard derivate direttamente dalle domande d'esame note. Ottimizzate per lo studio rapido.
- `IUM_domande_flashcards_anki.tsv`: File tecnico per l'esportazione automatica massiva su software come Anki. Formato: ID - Domanda - Risposta - Tags.
- [[ium_flashcards]]: Il vecchio blocco di flashcards generali create gradualmente durante l'importazione degli appunti. Utili come extra.
