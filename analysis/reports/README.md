# Reports di Analisi

Questa cartella contiene un report per ogni asset analizzato.

Ogni report deve:

- riferirsi a un solo asset;
- indicare path, tipo e stato dell'analisi;
- estrarre concetti, definizioni, esempi e domande;
- segnalare conflitti o dubbi;
- indicare le azioni richieste a Codex.

La knowledge base in `knowledge_base/` può essere aggiornata solo a partire da questi report.
